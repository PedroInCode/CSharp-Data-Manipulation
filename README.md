# 📊 C# Data Manipulation: Coleções, LINQ e Serialização

> **Status: Em Desenvolvimento 🚀**
> 
> Este repositório contém meus estudos aprofundados sobre a manipulação de dados no ecossistema .NET. O objetivo é dominar desde as coleções básicas até técnicas avançadas de filtragem com LINQ e troca de dados via JSON.

---

## 📂 Estrutura da Solução

A solução está dividida em projetos numerados, facilitando o acompanhamento do progresso:

1. **01-IntroAColecoes**: Diferença entre `ArrayList`, `Arrays` e o poder dos `Generics <T>`.
2. **02-OperacoesEmColecoes**: Métodos de manipulação, ordenação e busca.
3. **03-AbstraindoAFonteDeDados**: Preparando dados para consultas complexas.
4. **04-LINQ**: O "Language Integrated Query" — filtragens e projeções de nível sênior.
5. **05-Strings**: Manipulação avançada de textos e formatação.
6. **06-ExpressoesRegulares (Regex)**: Validação de padrões (E-mails, CPFs, etc).
7. **07-Serializacao**: Transformando objetos em JSON (A base da comunicação em APIs).

## 🧠 Principais Aprendizados

### 🛡️ Generics & Segurança
Entendimento de como as listas genéricas (`List<T>`) evitam erros em tempo de execução ao restringir os tipos de dados permitidos em uma coleção.

### 🪄 LINQ (Language Integrated Query)
Aplicação de sintaxe declarativa para realizar operações de banco de dados diretamente em coleções C#, tornando o código mais legível e performático.

### 🔄 Serialização JSON
Essencial para o desenvolvimento de APIs, aprendendo a trafegar dados entre o Back-end e sistemas externos.
