# 🛠️ C# Data Manipulation: Coleções, LINQ e Expressões Regulares

Este repositório contém os projetos práticos e estudos desenvolvidos durante a formação de C# da Alura. O foco principal é o domínio de manipulação de dados, desde estruturas básicas de coleções até consultas complexas e processamento de strings.

## 🗂️ Estrutura da Solução

A solução está organizada em múltiplos projetos de console, cada um abordando um pilar específico da linguagem:

- **01-IntroducaoAColecoes**: Fundamentos de arrays, `IEnumerable`, `IEnumerator` e os bastidores do `foreach`.
- **02-OperadoresEmColecoes**: Ordenação, filtros e manipulação de listas personalizadas.
- **03-AbstraindoAFonteDeDados**: Práticas de design de código e separação de responsabilidades.
- **04-LINQ**: Consultas poderosas em coleções usando a sintaxe de consulta e métodos de extensão.
- **05-Strings**: Manipulação avançada de textos e formatação.
- **06-ExpressoesRegulares**: Validação de padrões usando Regex.
- **07-Serializacao**: Conversão de objetos para JSON e XML.

---

## 🚀 Projetos de Prática (Autorais)

Além dos exercícios das aulas, este repositório contém implementações próprias para fixação de conceitos:

- **x02-Pratica-POO**: Aplicação dos conceitos de Manipulação de Dados integrados com os pilares da Programação Orientada a Objetos (POO).
  - **Destaques:** Uso de `Namespaces`, `Encapsulamento` (modificadores de acesso), `Construtores` e implementação completa da interface `ICollection<T>`.
